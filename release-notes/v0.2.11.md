# GrommeR v0.2.11

**Předchozí verze:** v0.2.10

## Hlavní změny

- Oprava CI: `TAURI_SIGNING_PRIVATE_KEY` u kroku `tauri build` (vyžadováno při `createUpdaterArtifacts: true`).
- Linux: `appimage` zpět v `bundle.targets` – AppImage se znovu generuje a nahrává na release.
- Kanonické ověřené postupy v `.cursor/rules/verified-procedures.mdc` (release + velká PDF).

## Opravy

- Produkční tagy v0.2.9/v0.2.10 neměly funkční GitHub Release buildy (signing při buildu, chybějící AppImage).

## Poznámky pro instalaci

- Windows: NSIS/MSI nebo `GrommeR_portable.exe`
- macOS (Apple Silicon): DMG
- Linux: AppImage
- Automatická aktualizace: instalované verze přes `latest.json`
